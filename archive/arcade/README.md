# Archived Arcade Authoring Tools

The browser Mode Forge editor and `LevelPackage` import/export pipeline were
archived on 2026-07-16. They are not served by the arcade runtime and are not
part of the active test suite.

Current game modes are authored directly in `arcade/levels.json`, their map
JSON files, and the Python runtime. AI-assisted iteration is the preferred
workflow for the Open Sauce build.
