A reconfigurable labyrinth maze.

Overview

TiltyTable is an open-source hardware/software project for a physical labyrinth game. The current repo contains early control code, computer vision experiments, Arduino firmware, and prototype media.

System

  • The maze surface tilts on a 3-axis Stewart-style platform.
  • The player input is an arcade roller ball, not a joystick.
  • An Azure Kinect looks down at the board for depth and game-state tracking.
  • The maze cells are being designed to change state: floor, wall, trap, or hole.

Current status

  • Stewart platform motion tests are running on the bench.
  • Azure Kinect color/depth capture and ball tracking code exists.
  • Roller-ball input is wired into the control path as a USB input device.
  • Reconfigurable maze-cell mechanisms are still being designed and tested.
